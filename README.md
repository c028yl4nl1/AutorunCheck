# AutorunCheck


Essa ferramenta automatiza o processo de brute force dos seguintes protocolos a partir de uma combolist:

- Ftp 
- Wordpress
- CPanel
- Mysql
- Mongdb 


Format: url:user:pass or https://url:port:user:pass
